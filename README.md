# Klasörü Yapısı

# Docker Başlatma
Docker'ı başlatmak
* docker-compose up -d

Çalışan Docker'ın içine girmek için
* docker exec -it internship_project_container bash

Çalışan Docker'ları listelemek
* docker ps

Açık olan Docker'ları kapatmak için
* docker compose down
